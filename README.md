# GBDA302 Week 6 – Sidequest

A pixel-art platformer built with [p5.js](https://p5js.org/) and [p5play](https://p5play.org/).

## Authors

Erin O'Neill

## Description

This game is an eight-bit style platformer which was inspired by the game werewolf which we played in class i changed the theme/background to be a forest in the nighttime. I tried making the villager the main character against the werewolf. I didnt exactly know what to make the collectables except coins which already existed so i made it a little blob character similar to the leaf. Then I changed the music to dark 8-bit style.

## Learning Goals

- learn how to use assets
- learn how to use sprite sheets
- lean how to implement parallax backgrounds
- learn how to implement music and attach them to actions

## Running the Game

Open `index.html` in a browser using a local web server (e.g. VS Code Live Server). Direct file:// access will not work due to browser security restrictions on ES modules and audio.
